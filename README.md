# AuraLearn--Interactive-AI-Tutor-platform-for-neurodiverse-students
This project is a full stack API based application working on thinking of gemini AI. It is personalised interactive learning platform for students having different neurological difficulites. The main feature of this application is AI chat agent which answers in a way that these students can grasp concepts with ease. Currently under development but the first version is out which is just a basic version of the project. Development is in progress
Major Technologies:
Frontend- React, Javascript,
Backend -Gemini API, Python.


AI tools used:
Cursor, Gemini, Loveable.
